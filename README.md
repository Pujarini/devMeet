# DevMeet

A place where you can find your Developers meetup at one place.

Networking, communities and opportunities

## Tech Stack

**Client:** React, Redux, Tailwind CSS , React router dom, multer, framer motion

**Server:** Node, Express, MongoDB

### Progress

7 days sprint ( Start date : 11 September)

- [ ] Meetup listing page (Homescreen)(12 September)
- [x] Login page for users (14 september wip)
